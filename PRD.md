# PRD — 반려견 배변 트래커 앱 (똥스키)

## 1. 문제 정의

- 견주는 반려견의 배변 패턴을 기억에 의존
- 변비 / 설사 / 이상 징후 판단이 늦어짐
- 병원 방문 시 배변 변화를 구체적으로 설명하기 어려움
- 반려견은 말을 못하기 때문에 배변이 건강 상태를 확인할 수 있는 핵심 창구

---

## 2. 타겟 사용자

- 반려견을 키우는 견주 (강아지 ~ 노견 전 연령)
- 다견 가구
- 바쁜 일상으로 건강 변화를 놓치기 쉬운 맞벌이 / 1인 견주
- 초보 견주 (정상 범위 판단 기준이 없는 경우)

---

## 3. 핵심 가치

> "10초 기록 → 우리 애 기준 자동 학습 → 조기 경고"

- 푸시 알림으로 기록을 유도하고, 기록 즉시 현황 확인
- **온보딩에서 베이스라인을 직접 입력** → 앱 설치 당일부터 개체 맞춤 경고 즉시 작동
- 병원 방문 시 리포트 내보내기로 수의사 진료 보조

## 6. 화면 구조

### 온보딩 플로우 (최초 1회, 목표 2분 이내)

```
스크린 1 — 반려견 이름 + 견종 + 나이        (~30초)
스크린 2 — 사이즈 + 체중                    (~15초)
스크린 3 — 중성화 여부                      (~5초)
스크린 4 — 평소 배변 색깔 선택              (~15초)
스크린 5 — 평소 배변 형태 선택              (~15초)
스크린 6 — 평소 하루 배변 횟수 선택         (~10초)
스크린 7 — 완료 + 첫 기록 유도              (~5초)
```

- 스크린당 결정 하나, 탭으로만 선택 (타이핑 최소화)
- 견종은 검색 + 자동완성 (직접 입력 가능)
- 체중은 숫자 입력, 사이즈는 소형 / 중형 / 대형 탭 선택 병행
- 온보딩 완료 즉시 경고 시스템 전체 활성화 (대기 기간 없음)
- 알림 권한은 첫 기록 완료 후 요청
- 나중에 설정 탭에서 모두 수정 가능

---

### 탭 구성 (3개)

| 탭 | 설명 |
|---|---|
| 📝 기록 | 앱 진입 시 첫 화면. 푸시 알림 → 바로 기록 |
| 📊 현황 | 마지막 기록 경과 + 주간 그래프 |
| ⚙️ 설정 | 반려견 프로필 / 알림 설정 / 배변 가이드 |

### 사용 흐름

```
푸시 알림 수신
  → 기록 탭 진입
  → 항목 탭 선택 (+ 선택적 메모 / 사진 첨부)
  → 기록 완료
  → 현황 탭으로 자동 이동
```

## 7. 반려견 프로필

> 아기 버전과 결정적으로 다른 부분. 프로필 정보가 정상 범위 자동 보정과 품종별 경고 활성화에 직접 사용됨.

| 항목 | 용도 |
|---|---|
| 견종 | 품종별 취약 질환 경고 자동 활성화 |
| 나이 | 강아지 / 성견 / 노견 기준 분기 |
| 체중 / 사이즈 | 정상 배변 횟수 범위 자동 보정 |
| 중성화 여부 | 호르몬 관련 소화 변화 참고 |

- 온보딩에서 입력한 모든 항목 여기서 수정 가능
- 다견 가구 지원: 반려견 추가 / 편집 / 삭제 (추가 시 동일한 온보딩 플로우 진행)

### 라이프스테이지별 정상 배변 횟수 기준

| 단계 | 나이 기준 | 정상 횟수/일 |
|---|---|---|
| 강아지 초기 | ~12주 | 5~6회 |
| 강아지 | 12주~6개월 | 3~4회 |
| 강아지 후기 | 6개월~1세 | 2~3회 |
| 성견 | 1세~7세 | 1~3회 |
| 노견 | 7세+ | 1~2회 (감소 가능) |

### 사이즈별 특성

| 사이즈 | 특성 |
|---|---|
| 소형견 (~10kg) | 신진대사 빠름, 배변 횟수 많음, 탈수 위험 빠름 |
| 대형견 (35kg+) | 배변 간격 길어도 정상, Bloat(위염전) 고위험 |

### 다견 전환 UX

- 전체 탭 상단 아바타 바 고정 (탭바와 별개)
- 선택된 반려견: 포인트 컬러(앰버) 테두리
- 아바타 우측 `+` 버튼: 반려견 추가 (온보딩 플로우 진행)
- 아바타 탭 → 해당 반려견 기준으로 기록 / 현황 전환
- 아바타 롱프레스 → 편집 / 삭제

## 9. 기능 상세

### 9-1. 기록 탭

탭(tap) 방식으로 6가지 항목 선택 후 저장

#### 기본 4C 항목

| 항목 | 선택지 |
|---|---|
| **형태 (Consistency)** | 5단계: 딱딱한 펠릿형 / 단단한 통나무형 / 부드럽고 찐득 / 묽고 쌓이는 형태 / 액체형 |
| **색깔 (Color)** | 갈색 / 노랑·주황 / 초록 / 검정(타르) / 빨강(선혈) / 흰색 / 회색·기름진 / 핑크·보라 |
| **코팅 (Coating)** | 없음 / 투명 점액 / 초록 점액 / 빨간 점액 |
| **내용물 (Content)** | 없음 / 풀 / 실·쌀알 모양 (기생충 의심) / 이물질 |

#### 추가 항목 (반려견 특화)

| 항목 | 선택지 |
|---|---|
| **양** | 적음 / 보통 / 많음 |
| **배변 행동** | 정상 / 힘주기 / 울거나 낑낑 / 걷다가 실수 / 자다가 실수 |

#### 선택적 기록

- **냄새 이상**: 평소와 다른 악취 여부 (토글)
- **메모**: 자유 입력 (간식 종류, 산책 시간, 스트레스 이벤트 등)
- **사진 첨부**: 수의사 상담용 (병원 방문 리포트에 포함)

- 배변 가이드 링크 제공 (설정 탭 내 페이지로 이동)
- 기록 완료 시 현황 탭으로 자동 이동

---

### 9-2. 현황 탭

**상단: 마지막 기록 경과**
- "마지막 기록으로부터 N시간 / N일 경과"
- 경고 상태에 따라 색상 표시 (초록 / 노랑 / 빨강)

**하단: 주간 그래프**
- 가로축: 최근 7일 날짜
- 세로축: 양 (적음 / 보통 / 많음)
- 기록한 날짜에 점으로 표시
  - 하루에 양이 다른 기록이 여러 개면 → 점 여러 개
  - 양이 같은 기록이 여러 개면 → 점 1개 + 숫자 표기
- 점 색깔: 정상(초록) / 주의(노랑) / 경고(빨강)
- 기록 없는 날: 빈칸 또는 회색 처리
- 점 탭 → 바텀시트 (기록 내용 + 수정 / 삭제)

**광고 배너**: 탭바 바로 위 고정 (AdMob)

---

### 9-3. 설정 탭

- **반려견 프로필 관리**: 추가 / 편집 / 삭제 (다견 지원)
- **알림 설정**: 시간 설정 + on/off (기본 오후 9시)
- **배변 가이드 페이지**: 형태 5단계 / 색깔별 의미 / 4C 상세 설명
- **리포트 내보내기**: 기간 선택 → PDF 또는 이미지로 병원 공유
- **피드백 보내기**: Google Forms 연결

## 10. 푸시 알림

| 조건 | 메시지 |
|---|---|
| 기본 (사용자 설정 시간, 기본 오후 9시) | "오늘 {이름}이 볼일을 봤나요?" |
| 경고 상태 (무배변 초과 등) | "오늘 {이름}이 볼일을 못 봤어요. 확인해보세요." |
| 즉시 경고 항목 기록 시 | "{이름}의 배변 상태가 심상치 않아요. 수의사 방문을 고려해보세요." |

- 다견 가구: 선택된 반려견 기준으로 각각 알림 발송
- 알림 권한은 첫 기록 완료 후 요청 (온보딩 중 요청 안 함)
- 경고 알림은 기록 저장 직후 즉시 발송

## 11. 경고 시스템

> 아기 버전과 달리, 절대 기준 + 개체별 베이스라인 대비 변화 두 가지를 함께 사용

### 즉시 경고 (병원 방문 권고)

| 조건 | 근거 |
|---|---|
| 색깔: 검정(타르) / 빨강(선혈) | 소화기 출혈 의심 |
| 색깔: 흰색 | 간·담관 문제, 기생충 감염 의심 |
| 색깔: 핑크·보라 + 라즈베리잼 형태 | 출혈성 위장염(HGE) 의심 |
| 코팅: 빨간 점액 | 감염·출혈 의심 |
| 내용물: 실·쌀알 모양 | 기생충(촌충 등) 감염 의심 |
| 내용물: 이물질 | 장폐색 위험 |
| 강아지 24시간 무배변 | 강아지는 변비도 응급 |
| 성견 48시간 무배변 + 힘주기 | 변비 / 장폐색 의심 |
| 대형견: 식후 복부 팽창 + 무배변 | Bloat(위염전) 응급 |

### 주의 (48시간 내 경과 관찰, 지속 시 병원)

| 조건 | 근거 |
|---|---|
| 색깔: 노랑·주황 지속 | 간·담낭·췌장 의심 |
| 색깔: 초록 반복 | 담낭 / 기생충 / 풀 과다 |
| 색깔: 회색·기름진 | 지방 흡수 장애, 췌장 문제 |
| 코팅: 초록 점액 | 감염 가능성 |
| 형태 4~5단계 묽은 변 반복 | 설사 의심 |
| 베이스라인 대비 급격한 횟수 증가 | 소화 이상 / 스트레스 |
| 베이스라인 대비 급격한 횟수 감소 | 변비 / 식욕 저하 연계 가능 |
| 배변 행동 이상 반복 (힘주기, 낑낑) | 통증 / 폐색 가능성 |
| 노견 실내 실수 (걷다가 / 자다가) | 신경계 이상 / 인지기능장애 가능성 |

### 품종별 추가 경고 (프로필 기반 자동 활성화)

| 견종 | 추가 경고 |
|---|---|
| 저먼 셰퍼드 | 복부 팽창 + 무배변 → 즉시 경고 강화 |
| 그레이트 데인 | 식후 30분 내 무배변 + 팽창 → Bloat 즉시 경고 |
| 박서 | 점액변 반복 → 주의 알림 |
| 미니어처 슈나우저 | 회색·기름진 변 → 췌장염 경고 강화 |

## 12. 베이스라인 시스템

> 온보딩에서 직접 입력받아 설치 당일부터 개체 맞춤 경고 즉시 작동

### 온보딩 입력값 → 경고 기준 매핑

| 온보딩 입력 | 경고 시스템 활용 방식 |
|---|---|
| 견종 + 나이 + 사이즈 | 라이프스테이지 및 품종별 경고 즉시 활성화 |
| 평소 색깔 | 이 색깔이 기록되면 경고 안 함, 다른 색깔이면 주의 판단 |
| 평소 형태 | 평소 단계 ±2단계 이상 변화 시 주의 |
| 평소 하루 횟수 | 입력값 대비 2회 이상 증감 시 주의 |

### 베이스라인 업데이트

- 기록이 쌓이면서 패턴이 바뀐 경우 설정 탭에서 수동 업데이트 가능
- 추후: 기록 누적 데이터로 베이스라인 자동 보정 제안 (확장 기능)

## 13. 병원 연계 리포트

> 견주가 "겉으로 멀쩡해 보이는" 강아지의 변화를 수의사에게 정확히 전달하기 위한 기능

- 기간 선택 (7일 / 30일 / 전체)
- 포함 내용: 배변 그래프, 색깔·형태 변화 요약, 이상 기록 하이라이트, 첨부 사진
- 출력 형식: PDF / 이미지 공유
- 활용 시나리오: 동물병원 방문 전 미리 생성 → 진료 시 수의사에게 공유

## 14. 성공 지표

- 3일 이상 사용 지속률
- 온보딩 완료율
- 리포트 내보내기 사용률

| 형태 척도 | Bristol 1~7 | 5점 굳기 척도 |
