# strategy.md — 똥스키

## 서비스 개요

반려견의 배변을 10초에 기록하고, 변비·설사·이상 색깔을 자동 감지해 견주에게 조기 경고를 주는 모바일 앱.
응가뿡가(아기 배변 트래커)를 반려견용으로 변형한 파생 프로젝트.

## 타겟 유저

- **주 타겟**: 반려견을 기르는 초보 견주, 바쁜 맞벌이/1인 가구 견주
- **확장**: 다견 가구, 노견 보호자
- **특징**: 10초 기록 + 즉시 경고, 병원 방문 시 리포트 공유

## 핵심 플로우

```
푸시 알림 수신
  → 기록 탭 진입
  → 형태·색깔·코팅·내용물·양·행동 탭 선택
  → 기록 완료
  → 현황 탭 자동 이동 (경고 상태 즉시 반영)
```
